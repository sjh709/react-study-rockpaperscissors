## 1. 프로젝트 소개

- 컴퓨터와 하는 가위바위보 게임입니다.
- 가위바위보 버튼을 클릭하면 클릭한 값이 보입니다.
- 컴퓨터는 랜덤 한 값이 선택됩니다.
- 결과를 가지고 승패 결과를 보여줍니다.
- 승패 결과에 따라 테두리 색이 바뀝니다. (win: green, lose: red, draw: black)

<br>

## 2. 개발 환경
